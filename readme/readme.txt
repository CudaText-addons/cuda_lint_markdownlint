Linter for CudaLint.
Supports Markdown lexer.

Requires Node.js in PATH.
Uses "markdownlint-cli" Node package. You must install it from linter's folder
"(CudaText)/py/cuda_lint_markdownlint":
$ npm install markdownlint-cli

Linter doesn't support "global" installation.
